# ludiek

## 0.0.1

### Patch Changes

- ae36ed9: Let's begin!

  Today marks the start of _Ludiek_, a new and improved game engine the supersedes the
  previous [Incremental Game Template](https://github.com/123ishaTest/igt-library).

  It will be more awesome, more better, and a lot more work!

  The main difference is the support for completely static content.
  Where previously you would subclass built-in types to create custom behaviour, you now register concepts with the engine
  to handle this behaviour.
  Now any content can use this behaviour.

  Sounds vague, doesn't it? Well don't worry, the docs will make everything clear soon!

  Stay tuned :)
