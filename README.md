# Ludiek

> Ludiek is a game engine that focuses purely on the fun!

The project consists of the following parts:

- Ludiek, a collection of scripts to aid you with game development!
- Docs, documentation of the important parts
- Demo, an example game made with Ludiek
- Blog, biweekly updates on what's happening
- Content Studio, an interactive editor for your content

## Developing

Run the development server:

```bash
npm run dev
```

## Building

To create a static build of Ludiek

```bash
npm run build
```
